# 소비자 선호도 데이터 - 컬럼 설명

## 파일: `consumerpref/Consumer_Preference.csv`

OTT 플랫폼 이용률과 극장 방문 빈도 데이터 (연령대별, 2023-2024)

---

## 📊 데이터 개요
- **행 수:** 14개 (7개 연령대 × 2년)
- **기간:** 2023-2024년
- **데이터 출처:** 설문조사 기반

---

## 📋 컬럼 설명

### 기본 정보

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| `연도` | 조사 연도 | 2023, 2024 |
| `age` | 연령대 | 13-19세, 20대, 30대, 40대, 50대, 60대, 70대 이상 |

### OTT 플랫폼 이용률 (%)
**각 컬럼은 해당 연령대에서 그 플랫폼을 사용하는 사람의 비율을 나타냅니다.**

| 컬럼명 | 의미 |
|--------|------|
| `유튜브` | 해당 연령대의 유튜브 이용률 (%) |
| `넷플릭스` | 해당 연령대의 넷플릭스 이용률 (%) |
| `티빙` | 해당 연령대의 티빙 이용률 (%) |
| `웨이브` | 해당 연령대의 웨이브 이용률 (%) |
| `SOOP(구 아프리카TV)` | 해당 연령대의 SOOP 이용률 (%) |
| `카카오TV` | 해당 연령대의 카카오TV 이용률 (%) |
| `왓챠` | 해당 연령대의 왓챠 이용률 (%) |
| `쿠팡플레이` | 해당 연령대의 쿠팡플레이 이용률 (%) |
| `NAVER TV(구 NOW)` | 해당 연령대의 네이버TV 이용률 (%) |
| `디즈니플러스` | 해당 연령대의 디즈니+ 이용률 (%) |
| `U플러스모바일TV` | 해당 연령대의 U+모바일TV 이용률 (%) |
| `애플TV플러스` | 해당 연령대의 애플TV+ 이용률 (%) |
| `기타` | 해당 연령대의 기타 OTT 이용률 (%) |
| `OTT 비이용` | 해당 연령대의 OTT 미이용 비율 (%) |

### 극장 방문

| 컬럼명 | 설명 |
|--------|------|
| `theater_visit_freq` | 해당 연령대의 연평균 극장 방문 횟수 |

---

## 💡 데이터 해석 방법

### 예시: 2024년, 20대
```
넷플릭스: 57.6
유튜브: 86.0
극장 방문: 14.75
```

**의미:**
- 20대 중 **57.6%**가 넷플릭스를 이용함
- 20대 중 **86.0%**가 유튜브를 이용함
- 20대는 평균적으로 1년에 **14.75번** 극장 방문

---

## ⚠️ 주의사항

1. **중복 사용 가능:** 한 사람이 여러 플랫폼을 사용할 수 있어서 전체 합계가 100%를 초과할 수 있습니다.

2. **연령대 구분:**
   - 13-19세 = 10대
   - 20대 = 20-29세
   - 30대 = 30-39세
   - 40대 = 40-49세
   - 50대 = 50-59세
   - 60대 = 60-69세
   - 70대 이상 = 70세 이상

3. **극장 방문 빈도:** 숫자가 클수록 자주 방문함을 의미. ">" 표시는 최소값을 나타냄 (예: >9.25 = 9.25회 이상)
